Docker

Docker es un software que permite crear, probar e implementar aplicaciones lo que sucede con Docker es que empaqueta el software en unidades estandarizadas que son conocidas como contenedores como se vio en la página anterior, Docker es un sistema operativo para contenedores, de manera similar a como trabaja una vm pero este elimina la necesidad de administrarlas directamente (so guest), docker se instala en cada servidor y proporciona comandos sencillos (basados en GNU/UNIX) que pueden crear, iniciar y utilizar contenedores

Algunos beneficios

Docker trae consigo algunos beneficios como lo es la modularidad que es la capacidad de poder separar una aplicación para actualizar reparar actualizar o desactualizar sin la necesidad de que todo todo el servicio se tenga que detener.

las capas y control de versiones de imágenes se apoya de la tecnología GIT que como es bien sabido es un estándar en la industria a lo referente a control de versiones, en este caso docker utiliza capas para facilitar el diseño de nuevos contenedores.

Restauración es en parte de sus mayores beneficios ya que un contenedor cuenta con varias imágenes que pueden ser muy flexibles con las versiones de las imágenes y su restauración.

la implementación va de la mano de la restauración donde solo necesitas tener instalado el software de docker y descargar tu contenedor con todo definido y tu servicio ya puede estar en funcionamiento facilita teimpo, tecnologías y costos en temas de desarrollo, gestión, despliegue/producción, mantenimiento e incluso migración.
